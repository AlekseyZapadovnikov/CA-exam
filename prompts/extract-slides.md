# Преобразование слайдов в `slides.md`

Ты превращаешь материалы слайдов в понятный файл `sources/lecture-XX/slides.md`.

## Цель

Сохранить содержательную информацию со слайдов так, чтобы ее можно было использовать как источник определений, схем, таблиц, формул и картинок для экзаменационного билета.

## Источники

Используй:

- исходные слайды из `slides/lecture-XX/`;
- изображения страниц и схем из `assets/lecture-XX/`;
- автоматически извлеченный текст, если он уже есть в `sources/lecture-XX/slides.md`.

## Правила

- Сохранять определения.
- Сохранять схемы, таблицы, формулы и подписи.
- Если есть картинка или схема, вставить ссылку на файл из `assets/lecture-XX/`.
- Не выдумывать содержание слайдов.
- Не восстанавливать невидимый текст по памяти.
- Если слайд непонятен, пометить `[проверить слайд N]`.
- Если текст не извлечен автоматически, написать `[текст слайда не извлечён автоматически]`.
- Декоративные картинки не описывать как учебный материал.

## Формат для каждого слайда

```markdown
## Слайд N. Краткое название

![Slide N](../../assets/lecture-XX/slide-00N.png)

### Текст слайда

- ...

### Важные элементы

- определение: ...
- формула:

$$
...
$$

- схема: ...

### Комментарии для билета

- ...

### Проверить

- `[проверить слайд N]` ...
```

## Самопроверка

Перед сохранением проверь:

- все ли страницы слайдов представлены ссылками на изображения;
- не добавлены ли факты, которых нет на слайдах;
- не потеряны ли определения, таблицы, формулы и схемы;
- все ли проблемные страницы отмечены `[проверить слайд N]`.
